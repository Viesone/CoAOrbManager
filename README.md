Hides Necromancers Pet action bar frames while keeping the orb in WoW Ascension / Conquest of Azeroth

## Features

- ✅ Hides Necromancers Pet Action Bar with buttons
- ✅ Keeps Necromancers Pet Resource Orb visible
- ✅ Move orb anywhere on screen
- ✅ Resize orb (scale)
- ✅ Lock orb in place
- ✅ Lightweight
- ❌ Right-click tab no longer works (this is expected behavior since the orb is now detached from the action bar)

## Commands

| Command | Description |
|---------|-------------|
| `/orb move` | Make orb movable |
| `/orb lock` | Lock orb in place |
| `/orb reset` | Reset to bottom center |
| `/orb bottom` | Move to bottom center |
| `/orb top` | Move to top center |
| `/orb center` | Move to screen center |
| `/orb scale 1.5` | Change orb size (0.1-5.0) |
| `/orb hide` | Hide the orb |
| `/orb show` | Show the orb |
| `/orb status` | Show orb info |
| `/orb save` | Manually save settings |
| `/orb debug` | Show debug information |
| `/orb hideall` | Hide all CoA frames |
| `/orb showall` | Show all CoA frames |
| `/orb resetall` | Reset ALL settings to defaults |
| `/orb help` | Show this menu |

## Installation

1. Download the addon
2. Extract CoAOrbManager to `WoW Ascension\resources\ascension-live\Interface\AddOns`
3. Restart the game
4. Use `/orb` for commands
